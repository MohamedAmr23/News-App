## ğŸ¯ Key Features Explained

### Page Routing
Uses Next.js file-based routing system where each file in the `pages` directory automatically becomes a route, providing automatic code splitting and SEO-friendly URLs.

### Server-Side Rendering
News content is rendered on the server for better SEO, faster initial page loads, and dynamic content updates.

### Image Optimization
Leverages Next.js Image component for automatic optimization, lazy loading, and responsive images across different screen sizes.
